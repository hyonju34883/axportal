이미지 미리보기
변환 완료 후 여기에 마크다운 내용이 표시됩니다.

사용방법

  1. PPTX 파일 업로드
  2. 변환 시작 버튼 클릭
  3. 완료 후 마크다운 파일과 이미지 ZIP 다운로드

변환 방식

  1. LibreOffice로 PPTX → PDF 변환
  2. PDFBox로 PDF → 슬라이드 PNG 렌더링 (150 DPI)
  3. Apache POI로 텍스트/이미지 추출
  4. Claude Vision AI로 슬라이드 내용 분석
  5. RAG 최적화 마크다운 생성 + 이미지 ZIP 패키징
